## 🔍 Find What You Need

### By Topic
- **Data Tools**: NumPy, Pandas, Matplotlib
- **Statistics**: Descriptive, Distributions, Correlation
- **Regression**: Linear, Multiple, Gradient Descent
- **Classification**: Logistic, Decision Trees, SVM
- **Clustering**: K-Means, Hierarchical
- **Evaluation**: Metrics, Cross-validation, ROC-AUC
- **Ensemble**: Random Forest, Boosting, Stacking
- **Deep Learning**: Neural Networks, CNN, RNN
- **NLP**: Text Processing, Embeddings, Classification

### By Problem Type
- **Predict continuous**: Linear Regression
- **Predict category**: Logistic Regression, Decision Trees
- **Group data**: K-Means, Hierarchical Clustering
- **Complex patterns**: Neural Networks, Ensemble Methods
- **Analyze text**: NLP techniques
- **Optimize model**: Hyperparameter tuning, Ensemble

### By Dataset Type
- **Tabular**: Linear/Logistic Regression, Trees, Ensemble
- **Images**: CNN, Transfer Learning
- **Text**: NLP, Embeddings
- **Time Series**: ARIMA, LSTM, Prophet
- **Sequences**: RNN, LSTM
