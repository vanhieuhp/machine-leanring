# Phase 1 Project: Exploratory Data Analysis (EDA)

## Project Overview

This project teaches you how to perform a complete exploratory data analysis on a real dataset. You'll learn to:
- Load and explore data
- Clean and preprocess data
- Analyze distributions and relationships
- Create visualizations
- Generate insights

## Learning Objectives

By completing this project, you'll:
1. Understand the complete EDA workflow
2. Practice data cleaning techniques
3. Create meaningful visualizations
4. Draw insights from data
5. Build a foundation for ML projects

## Dataset

We'll use the Iris dataset - a classic dataset with:
- 150 samples
- 4 features (sepal length, sepal width, petal length, petal width)
- 3 classes (Iris setosa, versicolor, virginica)

## Steps to Complete

### Step 1: Load Data
- Load the dataset
- Check shape and structure
- Display first few rows

### Step 2: Explore Data
- Check data types
- Look for missing values
- Get statistical summary

### Step 3: Clean Data
- Handle missing values
- Remove duplicates
- Detect outliers

### Step 4: Analyze Distributions
- Visualize each feature
- Check for normality
- Identify skewness

### Step 5: Analyze Relationships
- Calculate correlations
- Create scatter plots
- Identify patterns

### Step 6: Generate Insights
- Summarize findings
- Identify important features
- Note data quality issues

## Expected Output

After completing this project, you should have:
1. A cleaned dataset
2. Statistical summaries
3. Visualizations showing distributions and relationships
4. A report with key insights

## Time Estimate

- Data loading and exploration: 30 minutes
- Data cleaning: 30 minutes
- Analysis and visualization: 1 hour
- Report writing: 30 minutes
- **Total: 2.5 hours**

## Tips

1. Start with `data_loader.py` to understand the data
2. Use `exploratory_analysis.py` for calculations
3. Create visualizations with `visualizations.py`
4. Document your findings as you go
5. Don't skip any step - each builds on the previous
