# Phase 3: Advanced ML Techniques - Learning Guide

## Overview

Phase 3 covers advanced algorithms and techniques that improve model performance and handle complex problems.

## Topics

### 1. Ensemble Methods
**What**: Combine multiple models for better predictions
**Algorithms**:
- Random Forest
- Gradient Boosting (XGBoost, LightGBM)
- AdaBoost
- Voting/Stacking

**When**: Need better accuracy, handle complex patterns

### 2. Support Vector Machines (SVM)
**What**: Find optimal hyperplane to separate classes
**Advantages**:
- Works well in high dimensions
- Effective with non-linear kernels
- Memory efficient

**When**: Binary classification, small to medium datasets

### 3. Neural Networks
**What**: Deep learning models inspired by brain
**Architectures**:
- Feedforward networks
- Convolutional networks (CNN)
- Recurrent networks (RNN)

**When**: Complex patterns, large datasets, images/text

### 4. NLP Basics
**What**: Process and analyze text data
**Techniques**:
- Tokenization
- Word embeddings
- Sentiment analysis
- Text classification

**When**: Working with text data
